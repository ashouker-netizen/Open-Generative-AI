Design: Etsy Pipeline Module

Section 3 of 3: Scheduler, Folder Saving & Notifications

📅 Scheduler (scheduler.js)

Since this is a browser-based app (not always running), the scheduler works like this:

  • On app open, checks localStorage for the last run date
  • If today's run hasn't happened yet and the time is past the configured hour (default 8am), it triggers the pipeline automatically
  • A "Run Now" button in the UI triggers it manually anytime
  • Only 1 run per day is allowed (enforced by the date check)

Note: the app must be open for the auto-run to fire. If you open it later in the day, it catches up and runs immediately.

📁 Folder Saving

Generated images are downloaded to a local folder using the browser's download API:

  • Filename format: etsy-YYYY-MM-DD-sleepy-elephant-stars.png
  • Default save location: your browser's Downloads folder (user can move it)
  • An "Open Folder" button in the UI opens the Downloads folder in Finder

🔔 Notification

Uses the browser's Web Notifications API (works in Electron/desktop apps):

  • Shows: "🎨 New Etsy print ready! Sleepy elephant under the stars — saved to Downloads"
  • Clicking the notification opens the app to the Etsy Pipeline tab
  • Requests notification permission on first use
