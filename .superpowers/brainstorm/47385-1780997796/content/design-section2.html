Design: Etsy Pipeline Module

Section 2 of 3: Idea Engine & Uniqueness Tracking

💡 Idea Engine (ideaEngine.js)

Generates a new nursery print concept each run. Combines randomized elements to produce a prompt for fal.ai:

Concept Formula
Subject (e.g. "sleepy elephant", "baby fox", "sunflower")
+ Theme (e.g. "under the stars", "in a cozy bed", "with alphabet blocks")
+ Style (e.g. "watercolor", "flat illustration", "soft pastel")
+ Format: "nursery wall art, 8x10, white background"

→ "Watercolor sleepy elephant under the stars, nursery wall art, soft pastel, 8x10"

The engine picks randomly from curated lists for each category. You can edit these lists from the UI settings panel.

🔒 Uniqueness Tracker (uniquenessDB.js)

Stored in localStorage under etsy_pipeline_history. Each entry records:

  • Concept string (subject + theme + style)
  • Date generated
  • Image filename
  • Status (approved / skipped)

Before generating, the engine checks: has this exact subject + theme combo been used before? If yes, it re-rolls until a unique combo is found (max 10 attempts). This guarantees no duplicate concepts in your shop.
